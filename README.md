
# Auto Progress




## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Hyagodesa/auto-progress.git
```

Entre no diretório do projeto

```bash
  cd auto-progress
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```

ou

```bash
  npm run dev
```
